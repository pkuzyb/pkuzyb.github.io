● A app for inspecting and editing articulator contour tracks (under development)


● A toolbox for gestural analysis based on articulator contour tracks (under development)

This tool allows the user to calculate the time series of displacement and velocity. Then, the user can create initial annotations, based on which an automatic algorithm is implemented to locate gestural landmarks (e.g., onset, peak velocity, etc.). The landmarks can be corrected manually by dragging the lines, and exported for statistical analysis.
